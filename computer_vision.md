# introduction
- **computer vision:** enable computers to analyse & understand images(or videos)
  - **low level:** work on pixel values  
    example: resizing, grayscale, edges, oriented gradients, color segmentation
  - **mid level:** connecting images to 3D  
    example: panorama stitching, multi-view stereo, optical flow, image stabilization
  - **high level:** interpreting meaning/content of images  
    example: image classification, object detection, semantic segmentation
