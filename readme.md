## build
- **windows:**
  ```bash
  cd .\build\windows\
  cmake ..\.. -G "Visual Studio 17 2022"
  ```
- **linux:**
  ```bash
  cd ./build/linux/
  cmake ../..
  make
  ./main_exe
  ```
